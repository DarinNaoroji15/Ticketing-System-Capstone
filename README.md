# Ticketing-System-Capstone

Creating a Ticketing System for a fictional company and it operates a standard Ticket System. For incoming tickets and outgoing ticket with their resolution. 

This project is a display of a ticketing system created through Command Prompt and running Visual Studio Code in the background. I was able to integrate AI to help build a base for the project.
